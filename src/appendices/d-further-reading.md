## Foundational Papers

### Distributed Systems Theory
- **The Part-Time Parliament** (Lamport, 1998) - The original Paxos consensus paper
- **In Search of an Understandable Consensus Algorithm** (Ongaro & Ousterhout, 2014) - Raft consensus, more accessible than Paxos
- **Harvest, Yield, and Scalable Tolerant Systems** (Fox & Brewer, 1999) - Practical interpretation of CAP theorem

### Data Systems Architecture
- **The Google File System** (Ghemawat et al., 2003) - Foundation of distributed file systems
- **MapReduce: Simplified Data Processing on Large Clusters** (Dean & Ghemawat, 2004) - The programming model that started it all
- **Resilient Distributed Datasets** (Zaharia et al., 2012) - The foundation of Apache Spark

### Table Formats
- **Delta Lake: High-Performance ACID Table Storage** (Armbrust et al., 2020)
- **Apache Iceberg: A Table Format for Huge Analytics** (Apache Foundation documentation)

### Stream Processing
- **The Dataflow Model** (Akidau et al., 2015) - Google's unified batch/streaming model
- **Streaming Systems** (Akidau et al., 2018) - Book-length treatment of stream processing
